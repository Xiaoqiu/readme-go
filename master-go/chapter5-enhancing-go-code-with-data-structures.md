## 关于图和点
- 循环图： 所有或者部分端点，在闭合的链路里面连接的。
- 非循环图： 没有闭合链路的。
- 有向图：边是有方向的
- 有向非循环图：是有向图，而且没有闭合链路的。
- 一般点事使用结果定义的，包含任何类型的信息。
